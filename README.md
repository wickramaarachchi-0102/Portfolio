# Responsive Personal Portfolio Website
Welcome to my personal portfolio website repository! This project is a multi-page, fully responsive web experience designed to showcase my academic background, technical skills, and personal journey as an IT and Management student.

Live Demo
You can view the live project here: https://github.com/wickramaarachchi-0102/Portfolio

 Project Overview
This portfolio consists of 5 core sections:
* Home:A welcoming hero section featuring a professional profile photo and a brief introduction.
* About: Details regarding my education at the University of Moratuwa, current location, and personal journey.
* Skills: A structured showcase dividing my technical and soft skills using a dynamic grid layout.
* Blog:Features external article links 
* Contact: A fully interactive contact form that allows users to send inquiries.
  

 Features & Technical Implementation
 1. Mobile-First Responsive Design
The entire layout adapts flawlessly to different screen sizes—from large desktop monitors to tablets and smartphones. 

 2. Dynamic State Changes (Form Submission)
The contact form uses JavaScript event handling to capture the `submit` event, prevent a default page reload using `e.preventDefault()`, and smoothly toggle CSS visibility classes to swap the form input fields with an instant confirmation message upon success.


Technologies Used
* HTML5: Structured semantic markup across all pages.
* CSS3: Custom layouts utilizing CSS Grid ,Flexbox, media queries, custom transition states, and modern pseudo-elements.
* JavaScript:Event listeners, class list manipulation, and asynchronous-style form state management.
